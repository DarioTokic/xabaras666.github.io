Slido is a web application that I developed as an assignment for my university coursework. It is a platform designed to allow lecturers to receive questions on different subjects and give responses in real-time. I developed this project using NodeJS, ejs, express, and PostgreSQL.

The application has a clean and intuitive user interface, which allows students to post questions, while the lecturers can log in to view questions and provide their responses. The platform allows for real-time communication, making it a valuable tool for educational institution.

The project is a testament to my proficiency in NodeJS, ejs, express, and PostgreSQL. The code is available on my GitHub page, and I continue to improve it by implementing new features.

Walk This Way is a web application that my colleagues and I developed during a university hackathon. The application is designed to help clubs and artists arrange gigs, while also allowing the public to find upcoming events they are interested in.

I was responsible for the frontend development of this project, including ejs, CSS, and JavaScript. In addition, I managed the project database, which was built using PostgreSQL. The frontend design is clean and intuitive, making it easy for users to find events.

The project is a testament to my proficiency in NodeJS, ejs, express, and PostgreSQL. The code is available on my GitHub page, and I continue to improve it by implementing new features.
